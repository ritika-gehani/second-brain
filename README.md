# AI Second Brain

AI Second Brain is a personal knowledge system inspired by Thiago Forte's *Building a Second Brain* methodology.

The goal of this project is to build an AI-powered system that helps users:

- capture knowledge
- retrieve information quickly
- revisit saved ideas
- generate insights and new ideas from stored knowledge

Instead of trying to remember everything you read or learn, a Second Brain allows you to store information externally and retrieve it when needed.

---

# Version 1 (MVP)

Version 1 focuses on a single core capability:

**Save knowledge and retrieve it later using semantic search.**

Users will be able to:

- save notes or articles
- store the content in a database
- search their saved knowledge using natural language

Example:

> "Find the article I saved about vector databases."

The system will return the most relevant saved content using vector similarity search.
