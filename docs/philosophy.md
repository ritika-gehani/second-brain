# AI Second Brain — Philosophy

## Overview

The AI Second Brain is inspired by the knowledge management framework introduced by Thiago Forte in *Building a Second Brain*.

The core idea is simple:

> Your brain should generate ideas, not store information.

Instead of relying on memory alone, information can be stored in an external system that helps you capture, organize, and retrieve knowledge when needed.

This project explores how modern AI tools can transform personal knowledge management into an intelligent system that helps users think, learn, and create.

---

# Core Philosophy

The system follows the **CODE framework**.

## Capture

Capture ideas and information that resonate with you.

Examples:

- articles
- blog posts
- research papers
- code snippets
- notes
- YouTube videos

Instead of letting information disappear, it becomes part of a system designed for retrieval.

---

## Organize

Information should be organized so that it can be discovered later.

Methods include:

- metadata
- tagging
- semantic embeddings
- structured storage

The goal is flexible discovery rather than rigid folders.

---

## Distill

Distillation extracts the most valuable ideas from stored knowledge.

Examples:

- summaries
- highlights
- key takeaways

This allows the system to surface important insights quickly.

---

## Express

The ultimate goal of knowledge management is **creation**.

Examples include:

- building projects
- writing blog posts
- synthesizing research
- generating new ideas

A Second Brain should help transform stored information into meaningful output.

---

# Vision

The AI Second Brain aims to become a system that helps users:

- store knowledge
- retrieve information quickly
- revisit previously saved content
- synthesize ideas
- generate new insights

In the long term, the system should act as:

- a personal research assistant
- a learning companion
- a creative idea generator

Instead of searching the internet for answers, users can search their own knowledge.
